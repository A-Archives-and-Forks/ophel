# 增强功能与设置

Ophel 在三个核心标签页之外，还提供了一系列覆盖页面交互、视觉定制、快捷操作和站点适配的增强功能。本页提供所有功能页面的导航索引。

## 页面增强

| 功能               | 说明                                                      | 文档                                       |
| ------------------ | --------------------------------------------------------- | ------------------------------------------ |
| 浮动按钮与工具箱   | 屏幕侧边的快捷按钮组和工具箱弹出菜单的行为与配置          | [浮动按钮](./enhancements/quick-buttons)   |
| 全局搜索           | Ctrl+K 统一搜索大纲、会话、提示词、设置，支持语法过滤     | [全局搜索](./enhancements/global-search)   |
| 布局与禅模式       | 自定义页面宽度、禅模式、面板尺寸与吸附行为                | [布局与禅模式](./enhancements/layout)      |
| 内容增强           | Markdown 渲染、LaTeX 复制、Mermaid 渲染、表格复制、去水印 | [内容增强](./enhancements/content)         |
| 阅读历史与滚动锁定 | 自动记录并恢复阅读位置；生成时锁定滚动位置                | [阅读历史](./enhancements/reading-history) |
| 用量统计           | 本地计数今日对话次数和估算 Token 用量                     | [用量统计](./site/usage-monitor)           |

## 外观与定制

| 功能       | 说明                                          | 文档                       |
| ---------- | --------------------------------------------- | -------------------------- |
| 外观与主题 | 12 套亮色 + 12 套暗色主题预设，支持自定义 CSS | [外观与主题](./appearance) |

## 标签页与通知

| 功能         | 说明                                               | 文档                                |
| ------------ | -------------------------------------------------- | ----------------------------------- |
| 标签页管理   | 自动同步对话标题、状态图标、隐私模式、新标签页打开 | [标签页与通知](./tab-notifications) |
| 生成完成通知 | 切换标签后 AI 生成完成时发送桌面通知和音效提醒     | [标签页与通知](./tab-notifications) |

## 站点配置

| 功能            | 说明                                                  | 文档                                 |
| --------------- | ----------------------------------------------------- | ------------------------------------ |
| 模型锁定        | 每次打开站点时自动切换到指定模型                      | [模型锁定](./site/model-lock)        |
| Claude 账号管理 | 存储多个 Session Key，达到限额时快速切换账号          | [Claude 账号管理](./site/claude)     |
| 站点专属设置    | Gemini 去水印、AI Studio 界面自动折叠、ChatGPT 修复等 | [站点专属设置](./site/site-settings) |
