# 全局搜索

![全局搜索叠加层 — 搜索框与分类结果](/images/enhancements/global-search.png)

按 **Ctrl+K**（Mac 上为 Cmd+K），或浮动按钮中的搜索图标，打开全局搜索叠加层。它把大纲节点、历史对话、提示词词库和设置项统一纳入一个搜索框，无需逐个标签翻找。

## 搜索范围

| 类型     | 说明                                       |
| -------- | ------------------------------------------ |
| 大纲节点 | 当前对话中所有标题与你的提问               |
| 会话     | 所有平台的历史对话（按标题、标签、文件夹） |
| 提示词   | 提示词库中的标题和内容                     |
| 设置项   | 设置页面中的功能名称和关键词               |

搜索结果按类别分组显示，用方向键上下导航，按 Enter 跳转，Esc 关闭。

## 语法过滤（会话搜索）

在搜索框中使用以下前缀，可以将结果精确限制到特定维度：

| 语法          | 作用                       | 示例          |
| ------------- | -------------------------- | ------------- |
| `folder:名称` | 只看指定文件夹内的会话     | `folder:工作` |
| `tag:名称`    | 只看带指定标签的会话       | `tag:研究`    |
| `site:平台`   | 只看来自某个 AI 平台的会话 | `site:claude` |

语法过滤与普通关键词可以联合使用，例如 `site:gemini tag:调研` 会找出 Gemini 平台中带"调研"标签的会话。

## 提示词搜索的插入行为

在搜索结果中选中一条**提示词**后按 Enter，默认行为随场景智能切换：

- 如果当前页面有可用的输入框，直接将提示词插入输入框
- 否则，跳转到提示词所在的标签页并高亮显示

可在 **设置 → 全局搜索** 中将此行为改为"仅定位"（只跳转高亮，不插入）。

## 触发方式

| 触发方式   | 说明                                                     |
| ---------- | -------------------------------------------------------- |
| `Ctrl+K`   | 默认快捷键（Mac 为 Cmd+K）                               |
| 浮动按钮   | 点击搜索图标                                             |
| 双击 Shift | 可选，在设置中开启；不同于其他快捷键，此触发无需聚焦页面 |

## 相关设置

**设置 → 全局搜索**：

| 设置项            | 说明                                                                    |
| ----------------- | ----------------------------------------------------------------------- |
| 双击 Shift        | 是否启用双击 Shift 触发搜索（适合不方便使用 Ctrl+K 的场景）             |
| 模糊搜索          | 开启后允许不完全匹配，搜索"转" 也能找到"翻转"等词；关闭则需精确子串匹配 |
| 提示词 Enter 行为 | 选中提示词结果后按 Enter 的动作：智能插入 or 仅定位                     |
