# 内容增强

这一组功能在 AI 对话页面上叠加额外的交互能力，包括对你的消息、对 AI 回复中的格式元素进行处理，让读写体验更顺手。

## 用户问题 Markdown 渲染

AI 输入框里写的 Markdown 语法（`**加粗**`、` ```代码块``` `、`- 列表` 等）发送后，消息气泡里通常只显示纯文本。启用此功能后，Ophel 会在你的消息气泡中渲染 Markdown 格式，让问题更便于阅读。

**设置 → 功能 → 内容 → 用户问题 Markdown 渲染**（默认开启）

## 双击复制 LaTeX 公式

AI 回复中包含数学公式时，鼠标双击公式渲染块，Ophel 会把对应的 LaTeX 源码复制到剪贴板，省去手动从源码查找的麻烦。

**设置 → 功能 → 内容 → 双击复制 LaTeX**（默认开启）

### 公式分隔符转换

复制 LaTeX 时，Ophel 可以自动转换分隔符格式：

- 行内公式 `$ ... $` ↔ `\( ... \)`
- 块级公式 `$$ ... $$` ↔ `\[ ... \]`

在 **设置 → 功能 → 内容 → 公式分隔符转换** 中开启。开启后，复制时分隔符统一转换为 `\( \)` / `\[ \]` 形式（如果你使用 Obsidian 或 Typora 等工具，此格式兼容性更好）。

## 一键复制 Markdown 表格

AI 回复中的表格会额外出现一个"复制"按钮。点击后，表格内容以标准 Markdown 表格语法复制到剪贴板，可直接粘贴到支持 Markdown 的编辑器中。

**设置 → 功能 → 内容 → 复制 Markdown 表格**（默认开启）

## Mermaid 图表兜底渲染

部分 AI 平台不会将 AI 回复中的 Mermaid 代码块渲染为图表——内容停留在代码文本形态。Ophel 在这类平台上会尝试自动将 Mermaid 代码渲染为图形，使图表能正常显示。

> 注意：此功能只在平台本身未渲染 Mermaid 时生效，不会干扰已有渲染结果。

**设置 → 功能 → 内容 → AI 回复 Mermaid 渲染**（默认开启）

## Markdown 修复（Gemini / AI Studio / ChatGPT）

Gemini、AI Studio 和 ChatGPT 在某些情况下存在**加粗文字无法正常渲染**的问题（`**文字**` 显示为纯文本）。Ophel 对受影响的平台提供修复补丁。

在各平台的站点专属设置中可以查看并开关此选项：

- **设置 → 站点设置 → Gemini → Markdown 修复**
- **设置 → 站点设置 → AI Studio → Markdown 修复**
- **设置 → 站点设置 → ChatGPT → Markdown 修复**

## 去水印（Gemini / AI Studio）

Gemini 和 AI Studio 在导出/截图某些内容时会附加"由 Gemini 创作"等水印。开启后，Ophel 会在导出时去除这些水印。

此功能依赖可选权限 `<all_urls>`（用于抓取图片并转为 base64）。首次启用时，浏览器会弹出权限请求，确认授权后生效。

**设置 → 站点设置 → Gemini → 去水印**

## 导出选项

**设置 → 功能 → 导出**：

| 设置项           | 说明                                             |
| ---------------- | ------------------------------------------------ |
| 自定义用户名称   | 导出文件中替代"我"的名称，留空则使用默认         |
| 自定义 AI 名称   | 导出文件中替代 AI 名称的文字，留空则使用平台名   |
| 文件名包含时间戳 | 导出时为文件名添加时间戳后缀，防止同名覆盖       |
| 包含思维链       | 导出时是否包含 AI 的推理过程内容（支持平台生效） |
