# 快捷键参考

所有快捷键均可在 **设置 → 快捷键** 中自定义或禁用。

下表中，**Alt** 在 Windows/Linux 上为 `Alt`，在 Mac 上为 `Option`；**Ctrl** 在 Windows/Linux 上为 `Ctrl`，在 Mac 上为 `Cmd`。

## 导航

| 操作           | 默认快捷键 |
| -------------- | ---------- |
| 跳转到对话顶部 | `Alt+T`    |
| 跳转到对话底部 | `Alt+B`    |
| 返回锚点位置   | `Alt+Z`    |

## 面板

| 操作                       | 默认快捷键 |
| -------------------------- | ---------- |
| 打开/关闭面板              | `Alt+P`    |
| 切换到第一个标签（大纲）   | `Alt+1`    |
| 切换到第二个标签（会话）   | `Alt+2`    |
| 切换到第三个标签（提示词） | `Alt+3`    |
| 切换亮色/暗色模式          | `Alt+D`    |
| 打开设置                   | `Alt+,`    |
| 查看快捷键列表             | `Alt+\`    |

## 如何自定义快捷键

1. 打开设置（`Alt+,`）
2. 进入 **快捷键** 页面
3. 点击任意快捷键行，然后按下新的组合键
4. 点击 **×** 可删除该快捷键
5. 页面顶部的开关可以一键禁用所有快捷键

你也可以配置**全局 URL**，以便通过浏览器扩展的快捷键设置页面将某个按键组合绑定到"打开指定页面"。
