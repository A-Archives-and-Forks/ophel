# 功能概览

Ophel 在 AI 对话界面上叠加了一个侧边面板，包含三个主标签页和多项页面增强功能。

## 三大主标签页

### 大纲 — 实时对话导航

对话进行时，Ophel 在大纲标签页中实时生成基于标题的树形结构。你可以：

- 点击任意条目，立即跳转到对应位置
- 滚动页面时，大纲自动同步高亮当前位置
- 按标题层级展开或折叠（`Alt+Shift+1~6`）
- 过滤只看"我的问题"或"我的书签"
- 鼠标悬停显示该节点的字数统计
- 为任意节点添加书签，方便二次定位

→ [查看大纲详细文档](./outline)

### 会话 — 管理所有对话

Ophel 将你在各 AI 平台的对话整合到统一列表，支持：

- 创建文件夹并将对话归类
- 添加标签（tag）进行多维标注
- 置顶重要对话
- 批量操作（移动、加标签、删除元数据）
- 导出对话（Markdown、JSON、纯文本）
- 通过 WebDAV 将对话元数据同步到私有服务器
- 全局搜索（`Ctrl+K`）按文件夹、标签、站点过滤

→ [查看会话详细文档](./conversation)

### 提示词 — 构建你的词库

把常用提示词保存起来，随时插入：

- 双击提示词直接插入对话输入框
- 支持变量占位符（`{变量名}`、`{{变量名}}`、`<变量名>`）
- 按分类组织，支持置顶
- 导入/导出提示词库（JSON 格式）
- 提示词队列：批量自动发送一组提示词，免去手动操作

→ [查看提示词详细文档](./prompt)

## 页面增强功能

除三个主标签页外，Ophel 还在 AI 页面上叠加多项增强：

### 浮动快捷按钮

一组可拖拽的浮动按钮，提供面板切换、全局搜索、主题切换、禅模式、滚动到顶部/底部、返回锚点等快捷入口。5 秒不操作自动折叠成胶囊形。

→ [浮动按钮与工具箱](../enhancements/quick-buttons)

### 幽灵穿透模式

按住 Ctrl/Command 约 0.2 秒，面板变透明并允许鼠标穿透点击背后内容。

### 面板吸附

拖动面板靠近屏幕边缘自动吸附；悬停边缘弹出预览；支持左右两侧。

→ [布局与禅模式](../enhancements/layout)

### 禅模式

隐藏侧边栏、顶部导航等无关元素，专注对话内容。

### 滚动锁定

生成中锁定滚动位置，防止被自动拖到底部，适合阅读长回复时使用。

### 阅读历史

记录每个对话的阅读位置，重新打开时自动滚动到上次阅读处。

→ [阅读历史与滚动锁定](../enhancements/reading-history)

### 高级模型用量统计

跟踪当天使用次数及估算 Token 量，并显示进度条。适合有每日配额限制的场景。

→ [用量统计](../site/usage-monitor)

### 提示词队列

设置好一组提示词，Ophel 自动逐条发送，每条等待 AI 完成后再发下一条。

→ [提示词库详情](./prompt)

### 模型锁定

每次打开指定站点时，自动切换到你选择的模型，无需每次手动操作。

→ [模型锁定](../site/model-lock)

### 内容增强

- 用户问题支持 Markdown 渲染（加粗、代码块、列表等）
- 双击公式复制 LaTeX 源码
- AI 回复表格添加一键复制按钮
- AI 回复 Mermaid 图表兜底渲染

→ [内容增强](../enhancements/content)

### 标签页管理

- 自动将浏览器标签页标题同步为对话标题
- 生成完成后发送桌面通知（含声音）
- 隐私模式：用自定义文字替换标签页标题

→ [标签页与通知](../tab-notifications)

### Claude 账号轮换

存储多个 Claude Session Key，当一个账号达到上限时，一键切换到下一个账号继续对话。

→ [Claude 账号管理](../site/claude)

### 主题与外观

12 套亮色主题 + 12 套暗色主题，支持自定义 CSS，带圆形扩散过渡动画。

→ [外观与主题](../appearance)
