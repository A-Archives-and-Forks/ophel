# 会话管理

![会话标签页 — 文件夹与标签列表](/images/features/conversation.png)

会话标签页整合了你在各 AI 平台的所有历史对话，提供统一的文件夹分类、标签系统、导出功能和云同步，让你彻底告别"找不到上次那个对话"的困境。

## 会话列表

Ophel 从当前 AI 平台的原生侧边栏读取会话列表，将其展示在统一列表中。你可以：

- 点击任意会话跳转到该对话页面
- 在搜索框输入关键词快速过滤（搜索范围包括标题、备注、标签）
- 查看每个对话的 Ophel 元数据：所属文件夹、标签、置顶状态
- 按修改时间、创建时间或标题排序

## 文件夹

将相关对话归入同一文件夹，实现按项目、主题或时间段的分类管理。

- 点击工具箱中的**移动到文件夹**为当前对话指定文件夹，或在会话右键菜单中操作
- 在会话列表侧边的文件夹栏中点击文件夹名称，过滤只显示该文件夹内的对话
- 文件夹支持 8 种颜色，方便视觉区分不同项目
- 可以将同一对话同时归入多个文件夹

## 标签（Tag）

标签是跨文件夹的多维分类方式，特别适合"一个对话属于多个维度"的场景（如既属于"工作项目"文件夹，也带有"调研"和"Python"标签）。

- 点击工具箱中的**设置标签**为当前对话添加或修改标签
- 在会话列表顶部的标签栏中点击标签名称快速过滤
- 标签名称完全自定义，无数量限制
- 多个标签之间可以组合过滤

## 置顶

将重要对话置顶后，它会固定显示在列表最顶部，不受排序规则影响。适合正在进行的项目或需要频繁回访的对话。

## 批量操作

长按（或右键）任意会话进入多选模式，框选后可以批量：

- 移动到文件夹
- 添加或修改标签
- 设置/取消置顶
- 删除 Ophel 中的元数据（不影响 AI 平台上的实际对话内容）

## 导出对话

点击工具箱中的**导出**，将当前对话内容导出为文件。支持三种格式：

| 格式         | 适用场景                                         |
| ------------ | ------------------------------------------------ |
| **Markdown** | 导入到 Obsidian、Notion 等笔记工具，格式保留完整 |
| **JSON**     | 程序化处理或长期存档，包含完整元数据             |
| **纯文本**   | 去除所有格式，最简洁，适合粘贴到其他文档         |

导出时可以精细控制内容范围：

- 是否包含你的问题
- 是否包含 AI 的回复
- 是否包含思考链（支持平台的推理过程展开内容）
- 是否包含元数据（时间戳、使用的模型名称等）

也可以点击工具箱中的**复制 Markdown**，直接将当前对话以 Markdown 格式复制到剪贴板。

## WebDAV 同步

在 **设置 → 备份与同步 → WebDAV** 中配置 WebDAV 服务器，将 Ophel 的**会话元数据**同步到你自己的私有服务器。

> 注意：WebDAV 同步的是 Ophel 在本地存储的元数据——文件夹归属、标签、置顶状态、备注等你在 Ophel 中添加的信息。**不包含 AI 对话的实际消息内容**，那些内容属于各 AI 平台，Ophel 无法导出。

支持手动同步（点击按钮）和按设定间隔自动同步两种模式。

## 常用快捷键

| 操作                 | 快捷键         |
| -------------------- | -------------- |
| 新建对话（新标签页） | `Ctrl+Shift+O` |
| 刷新会话列表         | `Alt+Shift+R`  |
| 定位当前会话         | `Alt+Shift+L`  |
| 上一个会话           | `Alt+[`        |
| 下一个会话           | `Alt+]`        |
| 导出当前对话         | `Ctrl+Shift+E` |
| 复制最新 AI 回复     | `Ctrl+Shift+C` |

## 相关设置

**设置 → 功能 → 会话**：

| 设置项         | 说明                                                        |
| -------------- | ----------------------------------------------------------- |
| 默认排序方式   | 按修改时间 / 创建时间 / 标题字母序                          |
| 显示站点徽标   | 在会话列表中每条旁显示对应 AI 平台的图标                    |
| 默认导出格式   | 点击导出按钮时优先使用的文件格式                            |
| 导出文件名模板 | 自定义导出文件的命名规则（支持 `{title}`、`{date}` 等变量） |
