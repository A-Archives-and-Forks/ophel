# 站点专属设置

各 AI 平台在页面结构和功能实现上各有差异，Ophel 为部分站点提供了专门的配置项。

进入 **设置 → 站点设置**，在左侧选择对应站点查看其专属选项。

## Gemini / Gemini Enterprise

**设置 → 站点设置 → Gemini（或 Gemini Enterprise）**

| 设置项        | 说明                                                              | 默认 |
| ------------- | ----------------------------------------------------------------- | ---- |
| Markdown 修复 | 修复 Gemini 偶发的加粗文字无法正常渲染的问题                      | 开启 |
| 去水印        | 去除导出/截图时附加的"由 Gemini 创作"水印；需要 `<all_urls>` 权限 | 关闭 |
| 主题跟随页面  | Ophel 面板主题自动跟随 Gemini 网页的亮色/暗色切换                 | 开启 |

**仅 Gemini Enterprise**：

| 设置项       | 说明                                                 | 默认 |
| ------------ | ---------------------------------------------------- | ---- |
| 策略重试     | 自动重试被企业内容策略拦截的请求（适合策略误报场景） | 关闭 |
| 最大重试次数 | 开启策略重试后，连续重试的最大次数                   | 3 次 |

## AI Studio

**设置 → 站点设置 → AI Studio**

这些设置控制 AI Studio 页面上各个区域的初始展开/折叠状态，帮助你减少每次打开时的手动整理操作：

| 设置项           | 说明                                            | 默认 |
| ---------------- | ----------------------------------------------- | ---- |
| 折叠左侧导航栏   | 加载时自动收起左侧项目导航栏                    | 关闭 |
| 折叠运行设置面板 | 自动折叠页面右侧的整个运行设置区                | 关闭 |
| 折叠工具面板     | 自动折叠运行设置中的"工具"子区域                | 关闭 |
| 折叠高级选项     | 自动折叠"高级设置"子区域                        | 关闭 |
| 默认开启搜索工具 | 进入 Prompt/Free 模式时自动勾选 Google 搜索工具 | 开启 |
| Markdown 修复    | 修复 AI Studio 中加粗文字无法正常渲染的问题     | 关闭 |
| 去水印           | 去除生成图片上的水印（需要 `<all_urls>` 权限）  | 关闭 |
| 默认模型         | 每次打开 AI Studio 时自动切换到的目标模型 ID    | 空   |

> **默认模型**（AI Studio 专属）与通用的**模型锁定**功能并存。AI Studio 的默认模型在此专属设置中配置，而不在模型锁定页。

## ChatGPT

**设置 → 站点设置 → ChatGPT**

| 设置项        | 说明                                          | 默认 |
| ------------- | --------------------------------------------- | ---- |
| Markdown 修复 | 修复 ChatGPT 偶发的加粗文字无法正常渲染的问题 | 关闭 |

## Grok / DeepSeek / Kimi / 豆包 / 元宝 & 其他

这些站点目前暂无专属设置项，通用的面板、布局、主题和快捷键功能均适用。
