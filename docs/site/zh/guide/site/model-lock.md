# 模型锁定

每次打开一个 AI 网站，你是否需要手动切回偏好的模型？模型锁定功能解决这个问题：设置好之后，每次页面加载时 Ophel 会自动为你切换到指定模型，省去重复操作。

## 工作原理

Ophel 在页面加载完成后，检测模型选择器的当前状态，如果不是锁定的目标模型，就自动触发切换。如果目标模型不存在（例如账号权限不包含该模型），Ophel 不会报错，保持当前模型不变。

## 配置步骤

1. 打开 **设置 → 站点设置**
2. 在左侧选择要配置的站点
3. 找到 **模型锁定** 部分，开启开关
4. 在输入框中填入目标模型的名称（关键词匹配，不区分大小写）

也可以通过工具箱菜单快速切换当前站点的模型锁定开关状态。

## 使用技巧

**关键词匹配**：输入模型名称的一部分即可，例如只填 `flash` 就能匹配 `Gemini 2.0 Flash`。如果多个模型都匹配，选第一个匹配到的。

**快速切换**：如果某天临时想用其他模型，直接在 AI 页面手动选择，Ophel 的锁定下次打开页面时才会生效，不会即时覆盖你的手动操作。

**按站点独立**：每个站点的模型锁定设置相互独立，可以为不同平台配置不同目标模型。
