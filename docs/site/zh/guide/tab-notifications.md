# 标签页与通知

Ophel 能接管浏览器标签页的标题显示，并在 AI 生成完成后主动通知你——即使你已经切到另一个标签页去做别的事。

## 标签页自动重命名

默认情况下，Ophel 会将浏览器标签页标题与 AI 对话标题保持同步，让你从标签栏一眼分辨多个对话窗口。

**设置 → 功能 → 标签页**：

| 设置项       | 说明                                                            | 默认值                     |
| ------------ | --------------------------------------------------------------- | -------------------------- |
| 自动重命名   | 是否同步对话标题到浏览器标签页                                  | 开启                       |
| 标题更新间隔 | 轮询检查标题变化的频率（秒）                                    | 3 秒                       |
| 标题格式     | 自定义标签页标题模板，支持变量 `{title}`、`{status}`、`{model}` | `{status}{title}->{model}` |
| 显示状态图标 | 在标题中显示旋转/对号图标（生成中/已完成）                      | 开启                       |

**在新标签页中打开**：勾选后，点击会话列表中的对话时，在新浏览器标签页中打开，而不是在当前页跳转。

## 生成完成通知

AI 在另一个标签页里完成回复时，Ophel 可以通过桌面通知和声音提醒你。

**设置 → 功能 → 提醒**：

| 设置项       | 说明                                                         | 默认值    |
| ------------ | ------------------------------------------------------------ | --------- |
| 完成通知     | 生成完成后发送桌面通知                                       | 关闭      |
| 通知音效     | 发送通知时是否同时播放提示音                                 | 开启      |
| 音效预设     | 内置多种音效可选                                             | softChime |
| 音量         | 通知音量（0–100%）                                           | 50%       |
| 重复次数     | 短音效重复播放的次数                                         | 3 次      |
| 重复间隔     | 每次重复之间的间隔（秒）                                     | 2 秒      |
| 已聚焦时通知 | 当前标签页已激活时是否也发送通知（适合专注某个长任务的场景） | 关闭      |
| 自动聚焦窗口 | 生成完成后自动将浏览器窗口置于前台                           | 关闭      |

> 首次启用"完成通知"时，浏览器会请求 **notifications** 权限，请选择"允许"。若之前拒绝过，可在 **设置 → 权限** 中补授。

## 隐私模式

使用 AI 时不想让标签页标题暴露正在讨论的内容？隐私模式会用一个自定义文字替换浏览器标签页标题，完全屏蔽真实内容。

**设置 → 功能 → 标签页**：

| 设置项   | 说明                               | 默认值 |
| -------- | ---------------------------------- | ------ |
| 隐私模式 | 开启后，标签页标题替换为自定义文字 | 关闭   |
| 隐私标题 | 替代显示的文字                     | Google |

> 隐私模式只影响浏览器标签页标题显示，不影响 AI 对话内容本身，也不影响 Ophel 面板内的任何功能。
